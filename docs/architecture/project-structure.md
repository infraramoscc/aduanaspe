# Project Structure & Architecture

## Directory Layout

```
.
├── docs/                   # Documentation root
│   ├── services/           # Service definitions & copy
│   ├── marketing/          # SEO strategy, campaigns, ad copy
│   ├── ui/                 # UI/UX, Design System, & Component specs
│   └── architecture/       # Technical guides (this folder)
├── src/                    # (Future) Application source code
└── public/                 # Static assets
```

## Maintenance Guide

### Updating Content
1. **Services**: Edit files in `docs/services/`.
2. **Marketing**: Edit files in `docs/marketing/`.
3. **UI/UX**: Edit files in `docs/ui/` for design system updates.

### Deployment
See `deployment.md` for build and deploy instructions.
