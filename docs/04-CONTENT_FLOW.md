# CONTENT_FLOW.md
## Guía de flujo de contenido y CTAs (referencial)
### Dominio raíz: aduanaspe.com

---

## 0. Alcance real del documento

Este documento **NO gobierna el desarrollo del blog**, ni impone
automatización de CTAs.

Su propósito es servir como **guía estratégica de referencia**
para que el responsable del contenido (humano) decida
**qué CTA colocar en cada artículo del blog**, y
**a qué página del dominio raíz `aduanaspe.com` dirigirlo**.

El blog:
- Se desarrolla aparte
- Se gestiona manualmente
- El CTA se decide caso por caso

El dominio raíz:
- Recibe tráfico ya educado
- Convierte estratégicamente
- No crea artículos de blog

---

## 1. Principio base (inalterable)

Aunque el CTA se elija manualmente, **el flujo conceptual no cambia**:

BLOG (externo, manual)
→ COMERCIO EXTERIOR (educa y filtra)
→ SERVICIOS (convierte)
→ CONTACTO (cierra)

Este documento existe para **no romper este flujo**, incluso con decisiones manuales.

---

## 2. Cómo decidir el CTA de un artículo del blog (regla humana)

Antes de elegir un CTA, el editor del blog debe responder:

1. ¿El lector aún está aprendiendo?
2. ¿Está comparando opciones?
3. ¿Tiene un problema real y urgente?

Según la respuesta, se define el destino del CTA.

---

## 3. Artículos informativos (Lead FRÍO)

### Perfil del lector
- Busca entender
- No sabe qué opción le conviene
- No está listo para contratar

### Ejemplos de temas
- ¿Qué es el agenciamiento de aduanas?
- ¿Cómo funciona una importación?
- ¿Qué documentos se necesitan para importar?
- Courier vs carga: diferencias generales

### CTA recomendado (manual)
**Destino:**
/comercio-exterior/*


Ejemplos:
- `/comercio-exterior/importacion`
- `/comercio-exterior/importar-desde-china`
- `/comercio-exterior/courier-vs-carga`

### CTA sugerido (copy)
- “Conoce cómo funciona este tipo de importación”
- “Descubre qué opción te conviene”
- “Revisa la guía completa”

### CTA que debes evitar
- ❌ Contacto directo
- ❌ WhatsApp
- ❌ Cotización

---

## 4. Artículos evaluativos / comparativos (Lead TIBIO)

### Perfil del lector
- Ya entiende el tema
- Compara alternativas
- Necesita orientación

### Ejemplos de temas
- Importar por courier o carga: ¿qué conviene?
- ¿Cuándo necesitas un agente de aduanas?
- Errores comunes al importar desde China
- Importación personal vs comercial

### CTA recomendado (manual)
**Destino principal:**
/comercio-exterior/*


**Desde allí:**
- CTA interno hacia `/servicios/*`

Ejemplo:
- Artículo → `/comercio-exterior/courier-vs-carga`
- CTA interno → `/servicios/agenciamiento-aduanas`

### CTA sugerido (copy)
- “Evalúa tu caso”
- “Ver el servicio adecuado”
- “Hablar con un especialista” (moderado)

---

## 5. Artículos de problema / urgencia (Lead CALIENTE)

### Perfil del lector
- Tiene un problema real
- Busca solución inmediata
- Tolera contacto directo

### Ejemplos de temas
- Qué hacer si SUNAT observa tu importación
- Mi carga está detenida en aduanas
- Multas frecuentes en importaciones
- Fiscalización aduanera: pasos a seguir

### CTA recomendado (manual)
**Destino directo:**
/servicios/*

Ejemplos:
- `/servicios/consultoria-aduanera`
- `/servicios/agenciamiento-aduanas`

### CTA permitido
- “Hablar con un especialista”
- “Evaluar mi caso ahora”
- WhatsApp externo (`wa.aduanaspe.com`)

---

## 6. Regla de simplicidad del CTA

- Un artículo → **un CTA principal**
- El CTA debe reflejar la etapa mental del lector
- No mezclar destinos

Ejemplo incorrecto:
- Mismo artículo enviando a:
  - Comercio Exterior
  - Servicios
  - Contacto
  - WhatsApp
