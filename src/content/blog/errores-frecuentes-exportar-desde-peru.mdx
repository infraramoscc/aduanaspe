## Exportar también puede generar contingencias

Muchas empresas se preparan mejor para importar que para exportar. Pero una exportación mal documentada también puede generar demoras, costos, rectificaciones, problemas de regularización o riesgos ante SUNAT.

El error de fondo suele ser el mismo: avanzar con la venta internacional sin ordenar producto, documentos, transporte, requisitos del destino y responsabilidades de cada actor.

## Error 1: vender sin confirmar requisitos del destino

No basta con que el producto pueda salir de Perú. También debe poder ingresar al país del comprador. Antes de cerrar la venta, revisa:

- requisitos sanitarios o fitosanitarios
- certificados de origen o calidad
- etiquetado o rotulado exigido
- documentos que pedirá el comprador
- restricciones o registros en destino
- condiciones de pago y entrega

Si el comprador exige documentos después de embarcar, el margen para corregir puede ser muy bajo.

## Error 2: usar un Incoterm sin entender el alcance

El Incoterm define quién asume costos, riesgos y responsabilidades. Una mala elección puede generar discusiones sobre flete, seguro, entrega, documentos o gastos en destino.

| Incoterm | Riesgo frecuente |
|---|---|
| EXW | El comprador maneja demasiados pasos y el exportador puede perder control documental |
| FOB | Deben coordinarse bien entrega, embarque y documento de transporte |
| CIF | El exportador asume flete y seguro hasta destino acordado |
| FCA | Requiere claridad sobre punto de entrega y transportista |

Si no tienes claro el alcance, revisa el contrato o la orden de compra antes de emitir documentos.

## Error 3: factura, packing list y carga real no coinciden

Este es uno de los problemas más comunes. La factura puede tener una descripción comercial, el packing list otra presentación y el documento de transporte un peso distinto.

Antes del embarque, cruza:

- descripción comercial
- cantidad
- unidad de medida
- peso bruto y neto
- bultos
- marcas
- datos del comprador
- Incoterm

La guía de [documentos para exportar desde Perú](/blog/documentos-para-exportar-desde-peru/) te ayuda a ordenar esta revisión.

## Error 4: dejar la regularización para el final

La exportación no termina cuando la carga sale. Después del embarque, la operación debe regularizarse con datos finales. Si nadie monitorea este punto, pueden quedar DAM pendientes, diferencias de peso o documentos incompletos.

Revisa también [regularización de DAM de exportación](/blog/regularizacion-dam-exportacion-sunat/) y [notificación de salida de mercancías](/blog/notificacion-de-salida-de-mercancias/).

## Error 5: no asignar responsables por etapa

En una exportación intervienen varias partes:

- exportador
- comprador
- agente de aduanas
- agencia de carga
- depósito temporal
- transportista
- entidad emisora de certificados

Si no está claro quién hace qué, aparecen vacíos. Por ejemplo: nadie pide el certificado, nadie verifica el peso final o nadie monitorea la regularización.

## Error 6: no guardar evidencia de la operación

Una exportación ordenada debe dejar rastro documental. Guarda:

- correos con comprador
- instrucciones de embarque
- cotizaciones
- factura final
- packing list final
- documento de transporte
- certificados
- constancias de transmisión

Esto ayuda si luego debes sustentar una diferencia, responder una observación o cerrar una auditoría interna.

## Señales de que debes pedir apoyo antes de exportar

Pide revisión si:

- es tu primera exportación
- el producto tiene requisitos sanitarios o sectoriales
- el comprador exige certificado de origen
- el Incoterm no está claro
- hay embarques parciales
- necesitas cobrar rápido después de regularizar
- ya tienes una DAM pendiente o una observación

## Qué hacer antes de embarcar

1. Confirma requisitos del destino.
2. Define Incoterm y responsabilidades.
3. Revisa factura, packing list y documentos de transporte.
4. Valida certificados o permisos necesarios.
5. Coordina con agente de aduanas y agencia de carga.
6. Define quién monitorea la regularización.

Si necesitas apoyo operativo, revisa [agenciamiento de aduanas](/servicios/agenciamiento-aduanas/) y [agencia de carga internacional](/servicios/agencia-de-carga-internacional/). Si el problema ya es una observación o riesgo de sanción, revisa [multas aduaneras SUNAT](/servicios/multas-aduaneras-sunat/).

## Matriz rápida de prevención

Usa esta lectura antes de confirmar una reserva, emitir factura final o entregar instrucciones de embarque:

| Riesgo | Señal de alerta | Acción recomendada |
|---|---|---|
| Requisito del país destino | El comprador pide un certificado, permiso o formato especial a última hora | Validar exigencias antes de producir o separar la carga |
| Diferencia documental | Factura, packing list y carga física no coinciden | Corregir antes del embarque y dejar trazabilidad |
| Incoterm mal entendido | Nadie sabe quién paga flete, seguro, almacenaje o destino | Confirmar responsabilidades por escrito |
| Regularización débil | El documento de transporte final llega tarde o con diferencias | Separar responsables y fecha límite de cierre |
| Evidencia incompleta | La operación depende de correos sueltos o archivos dispersos | Armar carpeta por operación con versiones finales |

Esta matriz no reemplaza la revisión técnica, pero ayuda a detectar cuándo la exportación necesita una mirada aduanera antes de que el problema llegue a SUNAT o al comprador.
