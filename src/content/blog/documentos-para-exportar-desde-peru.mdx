## Por qué los documentos definen el éxito de una exportación

Exportar no es solo conseguir un comprador internacional. La operación también debe sostenerse con documentos coherentes: producto, cantidad, valor, peso, condiciones de venta, destino, transporte y permisos aplicables.

Cuando los documentos no cuentan la misma historia, aparecen demoras, observaciones, rectificaciones o problemas para regularizar la operación después del embarque.

<InfoBox type="important" title="Revisión antes de embarcar">
  Lo más costoso no siempre es corregir un documento. Lo costoso es descubrir tarde que factura, packing list, documento de transporte y DAM no coinciden.
</InfoBox>

## Checklist base de documentos para exportar

Estos documentos suelen aparecer en una exportación formal:

| Documento | Para qué sirve | Qué revisar |
|---|---|---|
| Factura comercial | Sustenta la venta internacional | Comprador, valor, moneda, Incoterm, descripción y cantidad |
| Packing list | Detalla bultos, peso y presentación | Bultos, peso bruto, peso neto, marcas y embalaje |
| DAM de exportación | Declaración aduanera ante SUNAT | Partida, régimen, datos del exportador y mercancía |
| Documento de transporte | Prueba embarque y ruta | BL, AWB, guía o documento equivalente |
| Certificados | Sustentan origen, calidad o requisitos del destino | Vigencia, alcance, producto cubierto y entidad emisora |
| Permisos o vistos buenos | Aplica según producto | Entidad competente, momento de emisión y alcance |

No todos los embarques necesitan todos los documentos, pero sí necesitan una lógica documental clara.

## Factura comercial: más que un comprobante

La factura debe servir para que el comprador, el agente de aduanas, el transportista y SUNAT entiendan la operación. Revisa especialmente:

- razón social del comprador y exportador
- descripción comercial del producto
- cantidad y unidad de medida
- valor y moneda
- Incoterm acordado
- país de destino
- condiciones de pago
- relación con la orden de compra o contrato

Si la factura dice una cosa y el packing list otra, la exportación empieza con ruido.

## Packing list: donde nacen muchas diferencias

El packing list suele parecer simple, pero es clave para evitar diferencias de bultos, peso o presentación. Antes del embarque, valida:

- cantidad de cajas, pallets, sacos o unidades
- peso bruto y peso neto
- marcas o numeración de bultos
- tipo de embalaje
- coincidencia con factura y documento de transporte
- consistencia con lo que realmente se cargará

Una diferencia menor puede convertirse en un problema si afecta la DAM, el documento de transporte o la regularización.

## Certificados y permisos: dependen del producto y destino

Algunos productos necesitan certificados sanitarios, fitosanitarios, de origen, calidad, libre venta u otros documentos según el mercado de destino. También puede haber requisitos del comprador que no son aduaneros, pero sí comerciales.

Por eso conviene separar dos preguntas:

- qué exige Perú para permitir la salida
- qué exige el país de destino o el comprador para aceptar la carga

Para profundizar, revisa [certificado de origen para exportar desde Perú](/blog/certificado-origen-exportacion-peru/) y [errores frecuentes al exportar desde Perú](/blog/errores-frecuentes-exportar-desde-peru/).

## Cómo ordenar el expediente antes del embarque

Una carpeta de exportación debería organizarse por bloques:

### Bloque comercial

- cotización
- orden de compra o contrato
- factura comercial
- comunicaciones con el comprador

### Bloque logístico

- booking
- instrucciones de embarque
- packing list
- documento de transporte

### Bloque aduanero

- DAM
- mandato o autorización del agente
- canal asignado y actuados, si corresponde
- constancias de transmisión

### Bloque regulatorio

- certificados
- permisos
- autorizaciones
- documentos exigidos por destino

## Señales de alerta antes de exportar

Pide revisión antes de embarcar si:

- el producto tiene requisitos sanitarios o sectoriales
- el comprador exige certificados específicos
- factura y packing list no coinciden
- el peso final aún no está confirmado
- el producto se embarca parcialmente
- el Incoterm no está claro
- necesitas regularizar rápido para cobrar o cerrar la operación

## Relación con la regularización

Después del embarque, los datos finales deben sostenerse. Por eso esta guía se conecta con [regularización de DAM de exportación](/blog/regularizacion-dam-exportacion-sunat/) y [notificación de salida de mercancías](/blog/notificacion-de-salida-de-mercancias/).

Si ya tienes una observación o una diferencia documentaria, revisa [multas aduaneras SUNAT](/servicios/multas-aduaneras-sunat/). Si estás preparando una exportación formal, el servicio de [agenciamiento de aduanas](/servicios/agenciamiento-aduanas/) puede ayudarte a ordenar el despacho.
