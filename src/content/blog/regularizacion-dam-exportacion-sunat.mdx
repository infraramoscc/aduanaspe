## La exportación no termina con el embarque

Cuando la carga sale del país, todavía queda una etapa importante: regularizar la DAM de exportación con los datos finales de la operación. Este cierre permite que la declaración refleje lo efectivamente embarcado.

Si la regularización se descuida, pueden quedar operaciones pendientes, diferencias documentarias o riesgos para beneficios vinculados a la exportación.

<InfoBox type="important" title="Control posterior al embarque">
  Después del embarque, revisa si la notificación de salida fue transmitida, si el documento de transporte está final y si los datos de peso, bultos y fecha de embarque son consistentes.
</InfoBox>

## Qué se revisa al regularizar una DAM de exportación

La regularización normalmente exige comparar datos provisionales contra datos finales:

| Punto | Qué revisar |
|---|---|
| Documento de transporte | BL, AWB o documento final emitido |
| Peso | Peso ingresado, peso embarcado y peso declarado |
| Bultos | Cantidad real embarcada |
| Fecha de embarque | Fecha asociada a la salida efectiva |
| Factura | Datos finales de venta |
| Notificación de salida | Confirmación del depósito o responsable correspondiente |
| Series o parciales | Si toda la mercancía fue embarcada o quedó saldo |

La guía específica sobre [notificación de salida de mercancías](/blog/notificacion-de-salida-de-mercancias/) explica uno de los hitos clave de este proceso.

## Diferencias frecuentes después del embarque

### Diferencia de peso

Puede ocurrir por humedad, merma, redondeos, balanza, embalaje o diferencia entre peso comercial y peso operativo. Lo importante es revisar si la diferencia tiene explicación documental.

### Diferencia de bultos

Aparece cuando el packing list, el ingreso al depósito y el documento de transporte no coinciden. Antes de regularizar, conviene entender dónde nació la diferencia.

### Documento de transporte incompleto

Si el BL, AWB o documento equivalente aún no está final, la regularización puede quedar detenida o hacerse con datos que luego deben ajustarse.

### Embarque parcial

Cuando una operación se embarca parcialmente, hay que revisar si corresponde esperar el cierre completo, ajustar series, legajar lo no embarcado o tomar otra ruta según el caso.

## Documentos que conviene tener a la mano

Para revisar la regularización, ordena:

- DAM de exportación
- factura comercial
- packing list
- documento de transporte final
- booking o instrucciones de embarque
- ticket de balanza o constancia de ingreso, si aplica
- comunicaciones con agente, depósito, transportista o comprador
- certificados asociados a la operación

Si todavía estás antes del embarque, revisa [documentos para exportar desde Perú](/blog/documentos-para-exportar-desde-peru/).

## Riesgos de no regularizar bien

Una DAM pendiente o mal regularizada puede generar:

- seguimiento operativo innecesario
- observaciones internas o externas
- problemas para cerrar expediente
- dificultades para sustentar beneficios o devoluciones
- riesgo sancionador según el supuesto aplicable

No conviene trabajar con frases genéricas como "eso se arregla después". La regularización debe tener responsable, documentos y fecha de seguimiento.

## Qué revisar antes de enviar la regularización

Antes de cerrar la DAM, confirma que la información final tenga respaldo. Los puntos más sensibles suelen ser:

| Punto de control | Qué validar |
|---|---|
| Peso y bultos | Que coincidan con el documento de transporte, packing list y carga realmente embarcada |
| Documento de transporte | Que el número, fecha, nave/vuelo y datos del consignatario estén completos |
| Factura | Que corresponda a la mercancía embarcada y no a una versión preliminar |
| Embarque parcial | Que la DAM refleje solo lo que efectivamente salió |
| Certificados | Que los certificados usados no contradigan descripción, cantidad u origen |
| Comunicaciones | Que exista evidencia si hubo cambios de último minuto |

Si aparece una diferencia, no conviene regularizar de forma automática. Primero hay que entender si se trata de un ajuste normal de embarque, un error documental o una inconsistencia que puede convertirse en observación.

## Flujo recomendado de control

<StepProcess>
  <Step n={1} title="Confirma salida">
    Verifica que la carga efectivamente salió y que existe soporte del embarque.
  </Step>
  <Step n={2} title="Reúne documentos finales">
    Ordena factura, packing list, documento de transporte y constancias disponibles.
  </Step>
  <Step n={3} title="Cruza peso y bultos">
    Compara DAM, packing list, documento de transporte e información del depósito.
  </Step>
  <Step n={4} title="Detecta diferencias">
    Identifica si hay variación de peso, bultos, series, fecha o documento.
  </Step>
  <Step n={5} title="Cierra o escala">
    Si todo está consistente, regulariza. Si hay diferencias, define el sustento antes de transmitir.
  </Step>
</StepProcess>

## Cuándo pedir apoyo

Pide revisión si:

- la DAM sigue pendiente
- hay diferencia de peso o bultos
- el embarque fue parcial
- el documento de transporte cambió
- necesitas cerrar rápido la operación
- SUNAT o tu operador observó la regularización

Si la situación ya puede implicar una sanción, revisa [multas aduaneras SUNAT](/servicios/multas-aduaneras-sunat/). Para ordenar el despacho y seguimiento, revisa [agenciamiento de aduanas](/servicios/agenciamiento-aduanas/).
