## Que significa recibir una notificacion de SUNAT Aduanas

Recibir una **notificacion de SUNAT Aduanas** no siempre significa que ya tengas una multa. Puede tratarse de una solicitud de informacion, un requerimiento dentro de una fiscalizacion, una observacion documentaria, una comunicacion vinculada al despacho o una resolucion que ya contiene una consecuencia concreta.

El error mas comun es responder rapido sin entender tres cosas:

- que acto administrativo recibiste
- que plazo tienes para responder
- que supuesto tecnico esta detras de la comunicacion

Si respondes con documentos sueltos, sin ordenar el caso, puedes terminar debilitando tu posicion. Por eso el primer paso no es redactar: es **leer, clasificar y mapear el riesgo**.

<InfoBox type="important" title="Primero identifica el documento">
  No trates igual una esquela, un requerimiento, una carta, una resolucion de multa, una comunicacion de fiscalizacion o una duda sobre valor. Cada una puede exigir una respuesta distinta.
</InfoBox>

## Primeras 24 horas: que revisar antes de responder

Cuando llega una notificacion aduanera, conviene hacer una revision inicial el mismo dia:

1. Fecha de notificacion y medio por el que fue comunicada.
2. Numero de expediente, declaracion, DAM, regimen o documento asociado.
3. Area o intendencia que emite la comunicacion.
4. Hecho observado por SUNAT.
5. Plazo para atender, subsanar, presentar informacion o responder.
6. Consecuencia si no se atiende.
7. Documentos que SUNAT pide expresamente.
8. Documentos que no pide, pero que pueden ser utiles para explicar el caso.

Esta lista parece basica, pero evita un problema frecuente: contestar solo lo que se tiene a mano, sin revisar si la autoridad esta cuestionando valor, factura, descripcion, permisos, tributos, origen, clasificacion, regimen o consistencia documental.

## Tipos de notificaciones aduaneras que debes diferenciar

Una comunicacion de SUNAT Aduanas puede venir por distintas razones. Estas son las mas frecuentes desde la perspectiva del importador o exportador.

| Tipo de comunicacion | Que suele implicar | Riesgo practico |
|---|---|---|
| Requerimiento de informacion | SUNAT solicita documentos, explicaciones o datos | Responder incompleto puede abrir contingencia |
| Observacion documentaria | Hay inconsistencia entre factura, DAM, transporte u otros soportes | Puede terminar en multa o correccion |
| Fiscalizacion posterior | SUNAT revisa la operacion despues del despacho | Riesgo tributario, documentario o sancionador |
| Comunicacion sobre valor | Se cuestiona el valor declarado o su sustento | Puede conectar con ajuste de valor |
| Resolucion de multa | Ya existe una sancion determinada o comunicada | Puede requerir descargo o estrategia de impugnacion |
| Pedido por mercancia restringida | Se cuestiona permiso, autorizacion o documento sectorial | Riesgo de multa, retencion o problema operativo |

Si el problema esta en el **valor declarado**, revisa tambien [ajuste de valor aduanero](/servicios/ajuste-de-valor-aduanero/). Si el problema es mas amplio y ya hay notificacion, requerimiento o multa, la ruta correcta es [revision de multas aduaneras SUNAT](/servicios/multas-aduaneras-sunat/).

## Documentos que conviene ordenar

Antes de responder, arma una carpeta por operacion. Como base, incluye:

- notificacion recibida
- DAM o declaracion asociada
- factura comercial
- packing list
- documento de transporte
- seguro, si aplica
- documentos digitalizados transmitidos
- comunicaciones con proveedor, cliente, agente u operador
- pagos, transferencias, Swift, vouchers o estados de cuenta
- permisos, autorizaciones, certificados o VUCE si el producto es regulado
- resoluciones, rectificaciones o comunicaciones previas

No todos los documentos se presentan siempre. Pero revisarlos internamente permite decidir que conviene adjuntar, explicar o reservar para una respuesta mas tecnica.

## Como saber si la notificacion puede terminar en multa

Hay senales de alerta:

- SUNAT menciona un codigo o supuesto de infraccion.
- La comunicacion habla de documentacion incompleta, con errores, no veraz o no autentica.
- Se cuestiona valor, tributos, recargos o base imponible.
- La operacion involucra mercancia restringida.
- La factura esta incompleta, ilegible o no coincide con otros documentos.
- Se pide explicar una diferencia entre lo declarado y lo sustentado.
- Ya existe una resolucion o acto que determina sancion.

La [tabla de sanciones aduaneras SUNAT](/blog/multas-aduaneras-sunat-como-evitarlas/) debe leerse con cuidado porque cada supuesto tiene requisitos, infractor aplicable y sancion distinta.

## Errores comunes al responder una notificacion de SUNAT Aduanas

### Enviar todo sin orden

Mas documentos no siempre significan mejor respuesta. Si envias archivos contradictorios, incompletos o sin explicacion, puedes crear nuevas dudas.

### Contestar fuera de plazo

La fecha importa. Antes de redactar, confirma el plazo y evita dejar la respuesta para el ultimo momento.

### No revisar el supuesto tecnico

No es lo mismo un problema de valor que uno de factura, permisos, descripcion, regimen, origen o tributos. Cada supuesto exige una logica distinta.

### Copiar una respuesta generica

Una plantilla puede ayudarte a ordenar, pero no debe reemplazar el analisis del expediente. SUNAT revisa hechos, documentos y consistencia.

### No separar responsabilidad entre actores

Importador, exportador, agente de aduanas, agencia de carga, almacen, proveedor y transportista no cumplen el mismo rol. Identificar quien debia hacer que puede cambiar la lectura del caso.

## Que hacer si la notificacion se relaciona con valor declarado

Si SUNAT cuestiona precio, descuentos, pagos, factura, flete, seguro o condiciones de venta, el caso entra en terreno de **valoracion aduanera**.

En ese escenario, revisa:

- [ajuste de valor SUNAT](/blog/ajuste-de-valor-sunat-importacion/)
- [duda razonable aduanas SUNAT](/blog/duda-razonable-aduanas-sunat/)
- [documentos para sustentar el valor declarado](/blog/documentos-sustentar-valor-declarado-sunat/)
- [servicio de ajuste de valor aduanero](/servicios/ajuste-de-valor-aduanero/)

La diferencia es simple: si el problema principal es valor, esa landing especializada debe convertir. Si hay multa, requerimiento o fiscalizacion mas amplia, la pagina de [multas aduaneras SUNAT](/servicios/multas-aduaneras-sunat/) es la ruta comercial.
