## Qué significa recibir una notificación de SUNAT Aduanas

Recibir una **notificación de SUNAT Aduanas** no siempre significa que ya tengas una multa. Puede tratarse de una solicitud de información, un requerimiento dentro de una fiscalización, una observación documentaria, una comunicación vinculada al despacho o una resolución que ya contiene una consecuencia concreta.

El error más común es responder rápido sin entender tres cosas:

- qué acto administrativo recibiste
- qué plazo tienes para responder
- qué supuesto técnico está detrás de la comunicación

Si respondes con documentos sueltos, sin ordenar el caso, puedes terminar debilitando tu posición. Por eso el primer paso no es redactar: es **leer, clasificar y mapear el riesgo**.

<InfoBox type="important" title="Primero identifica el documento">
  No trates igual una esquela, un requerimiento, una carta, una resolución de multa, una comunicación de fiscalización o una duda sobre valor. Cada una puede exigir una respuesta distinta.
</InfoBox>

## Primeras 24 horas: qué revisar antes de responder

Cuando llega una notificación aduanera, conviene hacer una revisión inicial el mismo día:

1. Fecha de notificación y medio por el que fue comunicada.
2. Número de expediente, declaración, DAM, régimen o documento asociado.
3. Área o intendencia que emite la comunicación.
4. Hecho observado por SUNAT.
5. Plazo para atender, subsanar, presentar información o responder.
6. Consecuencia si no se atiende.
7. Documentos que SUNAT pide expresamente.
8. Documentos que no pide, pero que pueden ser útiles para explicar el caso.

Esta lista parece básica, pero evita un problema frecuente: contestar solo lo que se tiene a mano, sin revisar si la autoridad está cuestionando valor, factura, descripción, permisos, tributos, origen, clasificación, régimen o consistencia documental.

## Tipos de notificaciones aduaneras que debes diferenciar

Una comunicación de SUNAT Aduanas puede venir por distintas razones. Estas son las más frecuentes desde la perspectiva del importador o exportador.

| Tipo de comunicación | Qué suele implicar | Riesgo práctico |
|---|---|---|
| Requerimiento de información | SUNAT solicita documentos, explicaciones o datos | Responder incompleto puede abrir contingencia |
| Observación documentaria | Hay inconsistencia entre factura, DAM, transporte u otros soportes | Puede terminar en multa o corrección |
| Fiscalización posterior | SUNAT revisa la operación después del despacho | Riesgo tributario, documentario o sancionador |
| Comunicación sobre valor | Se cuestiona el valor declarado o su sustento | Puede conectar con ajuste de valor |
| Resolución de multa | Ya existe una sanción determinada o comunicada | Puede requerir descargo o estrategia de impugnación |
| Pedido por mercancía restringida | Se cuestiona permiso, autorización o documento sectorial | Riesgo de multa, retención o problema operativo |

Si el problema está en el **valor declarado**, revisa también [ajuste de valor aduanero](/servicios/ajuste-de-valor-aduanero/). Si el problema es más amplio y ya hay notificación, requerimiento o multa, la ruta correcta es [revisión de multas aduaneras SUNAT](/servicios/multas-aduaneras-sunat/).

## Documentos que conviene ordenar

Antes de responder, arma una carpeta por operación. Como base, incluye:

- notificación recibida
- DAM o declaración asociada
- factura comercial
- packing list
- documento de transporte
- seguro, si aplica
- documentos digitalizados transmitidos
- comunicaciones con proveedor, cliente, agente u operador
- pagos, transferencias, Swift, vouchers o estados de cuenta
- permisos, autorizaciones, certificados o VUCE si el producto es regulado
- resoluciones, rectificaciones o comunicaciones previas

No todos los documentos se presentan siempre. Pero revisarlos internamente permite decidir qué conviene adjuntar, explicar o reservar para una respuesta más técnica.

## Cómo saber si la notificación puede terminar en multa

Hay señales de alerta:

- SUNAT menciona un código o supuesto de infracción.
- La comunicación habla de documentación incompleta, con errores, no veraz o no auténtica.
- Se cuestiona valor, tributos, recargos o base imponible.
- La operación involucra mercancía restringida.
- La factura está incompleta, ilegible o no coincide con otros documentos.
- Se pide explicar una diferencia entre lo declarado y lo sustentado.
- Ya existe una resolución o acto que determina sanción.

La [tabla de sanciones aduaneras SUNAT](/blog/multas-aduaneras-sunat-como-evitarlas/) debe leerse con cuidado porque cada supuesto tiene requisitos, infractor aplicable y sanción distinta.

## Errores comunes al responder una notificación de SUNAT Aduanas

### Enviar todo sin orden

Más documentos no siempre significan mejor respuesta. Si envías archivos contradictorios, incompletos o sin explicación, puedes crear nuevas dudas.

### Contestar fuera de plazo

La fecha importa. Antes de redactar, confirma el plazo y evita dejar la respuesta para el último momento.

### No revisar el supuesto técnico

No es lo mismo un problema de valor que uno de factura, permisos, descripción, régimen, origen o tributos. Cada supuesto exige una lógica distinta.

### Copiar una respuesta generica

Una plantilla puede ayudarte a ordenar, pero no debe reemplazar el análisis del expediente. SUNAT revisa hechos, documentos y consistencia.

### No separar responsabilidad entre actores

Importador, exportador, agente de aduanas, agencia de carga, almacén, proveedor y transportista no cumplen el mismo rol. Identificar quién debía hacer qué puede cambiar la lectura del caso.

## Qué hacer si la notificación se relaciona con valor declarado

Si SUNAT cuestiona precio, descuentos, pagos, factura, flete, seguro o condiciones de venta, el caso entra en terreno de **valoración aduanera**.

En ese escenario, revisa:

- [ajuste de valor SUNAT](/blog/ajuste-de-valor-sunat-importacion/)
- [duda razonable aduanas SUNAT](/blog/duda-razonable-aduanas-sunat/)
- [documentos para sustentar el valor declarado](/blog/documentos-sustentar-valor-declarado-sunat/)
- [servicio de ajuste de valor aduanero](/servicios/ajuste-de-valor-aduanero/)

La diferencia es simple: si el problema principal es valor, esa landing especializada debe convertir. Si hay multa, requerimiento o fiscalización más amplia, la página de [multas aduaneras SUNAT](/servicios/multas-aduaneras-sunat/) es la ruta comercial.
