## Por que un error en la DAM puede ser delicado

La **Declaracion Aduanera de Mercancias (DAM)** concentra informacion clave de la operacion: mercancia, valor, regimen, tributos, documentos, datos del importador o exportador y otros campos relevantes.

Un dato incorrecto puede ser un error menor o puede abrir una contingencia seria. La diferencia depende de:

- que campo esta mal
- si afecto tributos o recargos
- si se relaciona con valor declarado
- si involucra permisos o mercancia restringida
- si fue corregido antes o despues de un requerimiento
- si SUNAT ya notifico una observacion o multa

Por eso no conviene minimizarlo con la frase "solo fue un error de tipeo" sin revisar el impacto.

## Campos que suelen generar riesgo

| Campo o dato | Riesgo posible |
|---|---|
| Valor | Ajuste, tributos omitidos, multa o duda razonable |
| Descripcion | Diferencia con factura, permiso, clasificacion o control |
| Partida arancelaria | Tributos incorrectos, permiso omitido o clasificacion observada |
| Cantidad, peso o bultos | Diferencia con factura, packing list o transporte |
| Factura | Documento incompleto, incorrecto o inconsistente |
| Regimen | Aplicacion equivocada de tratamiento aduanero |
| Origen | Riesgo en preferencias, certificados o TLC |
| Permisos | Mercancia restringida sin documento exigible |

Si el error ya fue notificado, revisa tambien [notificacion de SUNAT Aduanas](/blog/notificacion-sunat-aduanas-que-hacer/).

## Error con impacto tributario vs error formal

Una distincion importante es si el error dejo tributos o recargos sin pagar o garantizar.

Si el dato incorrecto afecta la base imponible, valor, partida, tasa o tratamiento tributario, el problema puede escalar. Si no afecta tributos, igual puede existir riesgo formal, dependiendo del supuesto.

En la [guia de multas aduaneras SUNAT](/blog/multas-aduaneras-sunat-como-evitarlas/) explicamos por que algunos supuestos distinguen entre errores con y sin tributos omitidos, y por que el momento de subsanacion puede importar.

## Que revisar antes de corregir o responder

Antes de tomar una accion, revisa:

1. DAM transmitida.
2. Factura comercial.
3. Packing list.
4. Documento de transporte.
5. Liquidacion de tributos.
6. Permisos o autorizaciones.
7. Sustento del valor declarado.
8. Comunicaciones con proveedor, agente u operador.
9. Si ya existe requerimiento o notificacion.
10. Si hubo rectificacion o subsanacion previa.

La correccion sin diagnostico puede no resolver el problema principal.

## Cuando el error en DAM se relaciona con valor

Si el dato incorrecto toca valor, precio, condiciones de venta, descuentos, pagos, flete, seguro o gastos asociados, el caso debe mirarse desde valoracion aduanera.

## Errores comunes al manejar una DAM con datos incorrectos

- Corregir sin evaluar si el error tuvo impacto tributario.
- No revisar documentos que originaron el dato.
- Presentar explicacion sin adjuntar soporte.
- No diferenciar error de valor, error documental y error regulatorio.
- Esperar una notificacion para recien ordenar el expediente.
- Asumir que el error es responsabilidad unica del agente de aduanas.

## Como prevenir errores en la DAM

La prevencion empieza antes de numerar:

- valida producto, uso, marca, modelo y descripcion
- revisa partida arancelaria con criterio tecnico
- confirma permisos si el producto es sensible
- alinea factura, packing list y transporte
- valida valor, flete, seguro y condiciones de venta
- no uses documentos preliminares como si fueran finales
- revisa la DAM antes de transmision cuando el caso sea relevante
