## Por que las mercancias restringidas generan tantas contingencias

Una **mercancia restringida** no esta prohibida necesariamente, pero requiere cumplir condiciones, permisos, autorizaciones, certificados o vistos buenos antes o durante el proceso aduanero.

El problema aparece cuando el importador compra, embarca o intenta despachar sin confirmar ese requisito. Para cuando SUNAT observa el caso, la carga ya puede estar comprometida y el costo de corregir sube.

La [guia de permisos para importar en Peru](/blog/como-saber-si-tu-producto-necesita-permisos-para-importar-en-peru/) ayuda a prevenir este riesgo antes de comprar. Esta pagina se enfoca en el escenario posterior: cuando ya hay observacion, notificacion o riesgo de multa.

## Que puede pasar si numeras sin documentacion exigible

Cuando se numera una declaracion de mercancia restringida sin contar con la documentacion exigible, puede aparecer un riesgo sancionador, ademas del problema operativo del despacho.

En la practica, el caso puede involucrar:

- falta de autorizacion o permiso
- documento sectorial obtenido fuera de momento
- producto mal identificado
- clasificacion arancelaria incorrecta
- error sobre si el control aplicaba o no
- documentos del proveedor insuficientes
- diferencias entre producto real y documento presentado

No basta con decir "no sabia". Hay que revisar si el producto estaba correctamente identificado, si el requisito era exigible y si existia alguna forma de sustentar la actuacion.

## Sectores donde conviene tener mas cuidado

| Sector | Riesgo frecuente |
|---|---|
| Salud, suplementos, cosmeticos | Registros, autorizaciones o controles sanitarios |
| Alimentos y contacto con alimentos | Requisitos sanitarios, rotulado o autorizaciones |
| Telecomunicaciones y electronicos | Homologacion, permisos MTC o documentos tecnicos |
| Juguetes y utiles | Rotulado, certificados, ensayos o control sanitario |
| Quimicos o productos peligrosos | Autorizaciones, fichas tecnicas, controles especiales |
| Seguridad y EPP | Normas tecnicas, certificados o autorizaciones |

Si tu producto esta en una categoria sensible, no conviene responder una observacion sin revisar clasificacion, uso, composicion, ficha tecnica y documentos sectoriales.

## Que documentos reunir

Para revisar el caso, ordena:

- notificacion o requerimiento recibido
- DAM
- factura comercial
- packing list
- documento de transporte
- ficha tecnica
- catalogo o descripcion del producto
- fotos o especificaciones
- partida arancelaria usada
- permisos, autorizaciones, registros o documentos VUCE
- comunicaciones con proveedor, agente u operador
- evidencia de fecha de obtencion del permiso, si existe

El punto central es probar que documento era exigible, cuando debia obtenerse y si la operacion cumplio o no con ese requisito.

## Errores frecuentes

### Comprar primero y preguntar despues

Es el error mas caro. Si el producto requiere autorizacion, el problema puede aparecer cuando la mercancia ya esta en transito o en deposito.

### Confiar solo en el proveedor

Que el proveedor haya vendido el producto a otros paises no significa que cumpla requisitos peruanos.

### Usar una partida arancelaria sin validar el control

La partida puede activar permisos o controles. Si la clasificacion es debil, todo el analisis se contamina.

### Presentar documentos que no corresponden al producto real

Certificados, fichas o registros deben coincidir con producto, modelo, composicion, lote o alcance aplicable.

## Como se conecta con multas aduaneras

La multa por mercancia restringida es solo una parte del problema. Tambien puede haber costos de almacenaje, demoras, rectificaciones, observaciones documentarias y perdida de margen.

Por eso conviene revisar el caso desde dos angulos:

- operativo: como destrabar o manejar la carga
- sancionador: como responder a SUNAT si ya hay notificacion o multa
